# 参与贡献

感谢参与字音岛。项目欢迎代码、课程内容校对、普通话发音建议、儿童体验反馈、无障碍改进和文档贡献。

## 开始之前

1. 搜索现有 Issue，避免重复提交。
2. 新功能或课程结构变更请先创建 Issue 说明目标、儿童使用场景和验收方法。
3. 不要提交儿童姓名、声音、照片、设备标识或其他个人信息。
4. 不要提交来源不清或未经授权的图片、音频、字体和课程材料。

## 开发流程

1. Fork 仓库并从 `main` 创建分支。
2. 使用 JDK 17 和 Android SDK 36。
3. 修改后运行：

```powershell
./gradlew.bat :app:testDebugUnitTest :app:assembleDebug
```

4. Pull Request 需说明问题、改动、验证结果和界面截图（如适用）。
5. 保持一次 PR 只解决一个清晰问题。

## 儿童产品要求

- 主要任务不能依赖儿童阅读说明文字。
- 每个新页面必须提供自动语音引导和可重复播放入口。
- 需要发音的内容应点击核心元素播放，而不是依赖页面底部点读条。
- 触摸目标、颜色对比、错误反馈和等待时间应适合 5—6 岁儿童。
- 新增拼音、汉字和词语内容须附普通话或幼教审核依据。
