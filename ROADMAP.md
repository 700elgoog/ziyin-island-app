# 路线图

## v0.6 — 开源基础与语音可靠性

- 完善所有页面的自动语音引导与核心元素点读测试。
- 替换或复核原型普通话音频，补齐清晰的授权与说话人信息。
- 增加无网络、拒绝录音权限和音频播放失败场景测试。
- 建立 GitHub Actions 自动构建和单元测试。

## v0.7 — 完整拼音课程

- 扩充声母、韵母、整体认读音节和拼读课程。
- 引入口型、舌位和气流的教学审核流程。
- 增加四声、听辨、拼读和纠错的分层练习。

## v0.8 — 识字与启蒙阅读

- 扩充高频汉字、词语和主题阅读内容。
- 建立字词复现与间隔复习机制。
- 为教师和家长提供本地学习报告导出。

## 长期方向

- 可替换的开放课程包格式。
- 更强的无障碍和低识字交互规范。
- 社区共同维护的普通话发音与课程质量检查工具。
