# 字音岛标准音录制任务书

## 录音目标

为 5—6 岁儿童提供稳定、自然、不过度夸张的普通话拼音标准音。最终素材将替换当前独立存放的 CC BY-SA 样板音频包。

## 人员要求

- 普通话一级乙等或以上，优先儿童教育配音经验。
- 女声或温暖中性声线，不使用角色腔替代标准发音。
- 课程老师现场或远程审听。

## 技术规格

- 48kHz、24bit、单声道 WAV 母带。
- 安静录音环境，峰值约 -6dBFS，不压限到失真。
- 每个音节前后保留约 200ms 安静区。
- 同一音节每个声调至少录制 3 条备选。
- 母带归档为 WAV；App 交付为 AAC/Opus，响度统一。

## 首批清单

`ā á ǎ à / ō ó ǒ ò / ē é ě è`

每条同时录制：

1. 单独标准音。
2. 较慢教学音。
3. 对应口型提示语。
4. 例字：大、马、爸、我、火、朵、鹅、河、乐。

## 审核

- 拼音老师确认声调、口型提示和发音自然度。
- 至少两台平板外放与耳机试听。
- 文件名、响度、静音长度和版权授权记录全部检查后入库。
