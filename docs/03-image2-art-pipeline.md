# Image2 美术资产生产规范 v0.1

## 目标

使用 Image2 高效生成丰富、现代、富有互动暗示的原创儿童画面，同时保证角色、色板、构图和版权边界可控。

## 视觉母版

- 世界观：海上的“字音岛”。
- 四个区域：声音森林、拼音工坊、汉字小镇、故事剧场。
- 向导角色：奶油色与珊瑚橙的小芽鸟，头顶叶片，圆润、友善、表情清晰。
- 色板：天空蓝、薄荷绿、暖奶油、珊瑚橙、向日葵黄、少量薰衣草紫。
- 材质：柔和 3D 黏土质感结合清晰 2D UI。
- 禁止：已有儿童教育品牌角色、伪中文、博彩式奖励、过度霓虹、细小按钮、恐怖或高压氛围。

首张母版：`design/image2-concepts/ziyin-island-home-v1.png`。

## 生产分层

1. 概念层：世界地图、主题场景、绘本关键帧。
2. 角色层：标准三视图、表情表、动作表、服装与比例锁定。
3. 互动层：前景物件、卡片、奖励物、场景热点。
4. UI层：背景和装饰可以使用生成资产；文字、按钮、进度和状态必须由 Compose 原生绘制。
5. 动画层：将筛选后的角色和物件拆分，再制作 Lottie 或骨骼动画，不直接播放整张 AI 图片序列。

## 一致性流程

1. 每个核心角色先确定角色卡和固定色值。
2. 后续生成均引用同一角色卡作为视觉参考。
3. 每批资产检查眼睛、手指、文字、边缘、光源和角色比例。
4. 通过的图像进入项目；未通过的图像只作为构图草案。
5. 所有教学文字均由程序渲染，不依赖图片内文字。

## 资产技术规格

- 平板场景母版：3:2 横屏，优先 3200×2136 或可安全裁切比例。
- 场景背景：WebP/AVIF，按低、中、高三档输出。
- 角色与互动件：透明 PNG/WebP，保留足够边距。
- 点击热点：另存结构化坐标，不把交互逻辑烘焙进图片。
- 文件命名：`区域_用途_对象_状态_v序号`，只使用英文小写和下划线。

## 当前母版提示词摘要

高质量 3:2 Android 平板首页，原创字音岛、四个学习区域、小芽鸟向导、两名儿童、柔和 3D 黏土与 2D UI 混合风格；仅允许“字音岛”“开始探索”两个中文短语；无广告、购物、排名、品牌模仿和水印。
