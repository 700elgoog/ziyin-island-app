# 儿童跟读与语音评测架构 v0.1

## 当前实现

- 麦克风权限只在儿童点击“开始录音”时请求。
- 录音采用 16kHz AAC/M4A，保存在应用缓存目录。
- 支持开始、停止、回听、重录和错误恢复。
- 只检测录音是否过短或过轻，不声称判断发音正确与否。
- 重录、退出课程或 ViewModel 释放时删除录音文件。
- 无麦克风设备仍可安装并完成其他课程。

## 为什么不使用系统 TTS 作为标准音

不同设备的中文 TTS 音色、声调、语速和离线资源不一致，不能作为儿童拼音教学的标准音。正式素材必须由普通话专业录音人员录制，并经过课程审校。

## 云端评测边界

`PronunciationEvaluator` 是供应商无关接口。未来腾讯云、讯飞或离线 SDK 都通过该接口接入，UI和课程逻辑不直接依赖供应商。

接入前必须完成：

1. 监护人单独同意流程。
2. 数据用途、保存期限和第三方接收方说明。
3. 原始录音删除策略。
4. 使用真实 5—6 岁儿童样本进行误判验证。
5. 低置信度不显示“错误”，只建议再次听读。
6. 供应商不可用时自动退化为录音回听，不阻塞课程。

## v0.2 进展

- 已接入 `a / o / e` 四声原型音频包；该包只用于验证，商业发布前必须替换或完成授权审查。
- 已增加录音实时音量动画。
- 已建立腾讯云拼音评测的临时凭证沙盒边界，尚未加入供应商 SDK 和网络权限。
- 在真机上验证麦克风、系统权限和音频焦点生命周期。
