# 字音岛完整 APP 重构说明 v0.3

## 为什么重构

上一版只能证明首页、单节 `a/o/e` 课程、录音和本地存储技术可行，不能作为完整儿童学习 APP。本次按“每个入口可进入、每节内容可学习、每次完成可记录”的标准重构。

## 完整页面

| 页面 | Image2 资产 | 主要功能 |
|---|---|---|
| 欢迎登岛 | `page_onboarding_v1` | 首次使用说明与开始入口 |
| 字音岛首页 | `ziyin_island_home_v1` | 四个学习区域、成长、家长门 |
| 声音森林 | `page_sound_forest_v1` | 24 首原创节奏儿歌、拍手互动 |
| 拼音课程地图 | `page_pinyin_map_v1` | 60 节分阶段课程与完成状态 |
| 拼音学习 | `page_pinyin_lesson_v1` | 认形、例字、标准音、跟读与回听 |
| 四声轨道 | `page_tone_game_v1` | 四声听辨与选择反馈 |
| 手指描写 | `page_tracing_v1` | 拼音和汉字触控描写 |
| 汉字小镇 | `page_literacy_town_v1` | 300 个汉字、主题筛选、学习状态 |
| 汉字学习 | `page_character_lesson_v1` | 字形、拼音、含义、组词与描写 |
| 复习乐园 | `page_review_games_v1` | 仅从已学内容出题，不提前显示答案 |
| 故事剧场 | `page_story_theater_v1` | 12 个原创三页短故事书架 |
| 故事阅读 | `page_story_reader_v1` | 分页阅读、前后翻页、完成记录 |
| 成长花园 | `page_achievements_v1` | 拼音、识字、故事三条成长进度 |
| 家长中心 | `page_parent_center_v1` | 时长、进度、陪伴建议和隐私说明 |

所有教学文字、拼音、汉字、按钮和进度均由 Compose 原生绘制，Image2 只负责场景和装饰，避免生成图片中的错字成为教学内容。

## 内容规模

- 60 节拼音短课：声音准备、6 个单韵母、23 个声母、18 个复/鼻韵母、16 个整体认读音节、拼读和规则复习。
- 300 个识字条目：60 个精编核心字和 240 个生活常用字。
- 24 首原创节奏儿歌。
- 12 个原创三页短故事。
- 64 个本地 MP3 文件，通过 81 条目标映射覆盖核心拼音学习页面。

## 发布前必须完成的专业工作

1. 240 个扩展汉字目前采用通用生活释义模板，正式课程需逐字补充课程人员审校的释义、组词、图像和例句。
2. 标准音仍来自 CC BY-SA 原型资源；正式商业版本必须换成自有专业录音或完成法务确认与授权履行。
3. 当前 12 个故事是短篇可读内容，不等同于完成插画分镜、配音和动画的出版级绘本。
4. 需要在小米 Pad 8 真机完成麦克风、扬声器、横竖屏、后台切换、性能和儿童可用性测试。
5. 腾讯云评测只有安全沙盒边界，未启用网络和供应商 SDK。

## 验收方式

运行 `scripts/verify.ps1`，会执行内容完整性测试、语音映射测试、进度测试、Android Lint 和调试 APK 构建。
