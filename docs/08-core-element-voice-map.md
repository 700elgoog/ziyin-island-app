# 字音岛 v0.5：核心元素点读与全页面语音清单

## 统一规则

1. 删除所有页面底部的全局语音点读栏，不再占用画面高度。
2. 页面打开时自动播报当前任务。
3. 需要重听或播放标准音时，点击正在学习的核心元素：拼音、汉字、口型、声调卡、词语、故事正文或成长卡。
4. 引导语音使用设备中文文字转语音；拼音标准音继续使用本地 MP3，不使用 TTS 冒充标准发音。
5. TTS、标准音和儿童录音互斥，新声音开始前停止另一条声音。

## 本次语音播放修复

- 在 Android 清单中声明 `android.intent.action.TTS_SERVICE`，满足 Android 11 及以上对文字转语音服务查询的要求。
- 引导语音改用媒体音量通道，和普通视频、音乐使用同一个音量设置。
- 依次尝试简体中文、中国中文、通用中文和设备默认语言。
- 中文语音包缺失、引擎启动失败或播放失败时，给家长显示明确错误提示。
- 点击标准拼音音频会停止 TTS；点击汉字、口型或故事语音会先停止标准音。
