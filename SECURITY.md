# 安全政策

## 支持范围

当前仅维护最新公开版本。早期版本可能不会获得单独补丁。

## 报告安全问题

请不要通过公开 Issue 披露儿童隐私、权限绕过、密钥泄露或可被利用的安全问题。请使用 GitHub 仓库的 **Security → Report a vulnerability** 私密报告功能；如果该功能暂不可用，请先创建一个不含漏洞细节的 Issue，请维护者开启私密沟通渠道。

报告应包含受影响版本、设备/Android 版本、复现条件、影响范围和建议修复方式。维护者会尽快确认并在修复发布后公开致谢（报告者要求匿名除外）。

## 密钥政策

APK 和仓库不得包含云服务长期密钥、签名私钥或服务账号文件。语音评测集成只能使用可信服务端签发的短期会话令牌。
