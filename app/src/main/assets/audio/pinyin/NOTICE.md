# Prototype audio notice

The files in this directory are temporary prototype pronunciation assets from:

- Project: `audio-cmn`
- Source: https://github.com/hugolpz/audio-cmn
- Syllable speaker: Chen Wang
- Curation and processing: Hugo Lopez, PLIDAM, INALCO
- License declared by the source repository: CC BY-SA

These files are kept as a separately versioned prototype pack and must not be represented as proprietary 字音岛 recordings. Replace them with professionally recorded and curriculum-reviewed first-party audio before a commercial release unless the final release deliberately complies with the applicable CC BY-SA attribution and share-alike obligations.

The core curriculum prototype also maps some isolated teaching targets to a conventional full syllable (for example, `b` uses `bo1`). These mappings are suitable for interaction testing only and require professional Mandarin teaching review before release.
